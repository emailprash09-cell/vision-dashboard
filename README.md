# AI Vision Dashboard
An advanced computer vision dashboard built with Python, MediaPipe, and TensorFlow.

## Features
* **Face Analysis**: Real-time emotion and facial landmark tracking.
* **Hand Tracking**: Precision gesture detection.
* **Pose Estimation**: Full-body movement analysis.
